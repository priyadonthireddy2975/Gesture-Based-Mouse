# Anti-Gravity Hand Gesture Mouse ✋🖱️

[![Python Version](https://img.shields.io/badge/python-3.10%2B-blue.svg)](https://www.python.org/)
[![Project Status](https://img.shields.io/badge/status-active-success.svg)](https://github.com/priyadonthireddy2975/Gesture-Based-Mouse)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A touchless interaction system using computer vision to control your computer's mouse with simple hand gestures. This project leverages MediaPipe and OpenCV to track hand landmarks and map them to screen coordinates, enabling a futuristic, "anti-gravity" feel.

---

## 🌟 Features

- **🚀 Smooth Tracking**: High-performance cursor movement using hand landmark detection.
- **🖱️ Pinch to Click**: Intelligent click detection by measuring the distance between the index finger and thumb.
- **🖼️ Fullscreen Support**: Adapts to any screen resolution for seamless interaction.
- **⚡ Real-time Processing**: Optimized for low latency using MediaPipe's efficient tracking models.

---

## 🛠️ Technologies Used

- **Python 3.10**: Core programming language.
- **OpenCV**: Image processing and camera handling.
- **MediaPipe**: Real-time hand landmark tracking.
- **PyAutoGUI**: Controlling mouse movements and clicks.
- **NumPy**: Efficient numerical operations.

## ⚙️ Setup Instructions

1. **Prerequisites**: Ensure you have Python 3.10 installed on your system.
2. **Clone the Repository**:
   ```bash
   git clone https://github.com/priyadonthireddy2975/Gesture-Based-Mouse.git
   cd Gesture-Based-Mouse
   ```
3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

---

## 🚀 How to Run

Run the application using the following command:

```bash
py -3.10 main.py
```

- **Move Mouse**: Move your index finger.
- **Click**: Pinch your index finger and thumb together.
- **Exit**: Press the `ESC` key on your keyboard.
